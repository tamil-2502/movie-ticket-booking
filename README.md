# 🎬 Movie Ticket Booking Application

## Overview

Movie Ticket Booking Application is an Android application developed using Kotlin and Android Studio. The app allows users to select a movie, choose seats, calculate ticket prices, confirm bookings, and view the final ticket details.

## Features

* View available movies
* Select a movie from the list
* Choose seats for booking
* Automatic ticket price calculation
* Booking confirmation screen
* Display final ticket details

## Technologies Used

* Kotlin
* Android Studio
* XML Layouts
* Android SDK

## Project Structure

* MainActivity – Displays movie list
* MovieDetailsActivity – Shows selected movie details
* SeatSelectionActivity – Handles seat selection and price calculation
* MainActivity2BookingActivity – Booking confirmation screen
* TicketActivity – Displays final ticket information

## How to Run

1. Clone the repository.
2. Open the project in Android Studio.
3. Sync Gradle files.
4. Run the application on an emulator or Android device.
